# demo

Demo footage of CoreSynq32 running on hardware.

## demo.mp4

Live demonstration showing:
- Programming a beat pattern across kick, snare, and hi-hat tracks
- Page toggle switching between steps 1–4 and 5–8
- BPM adjustment via potentiometer
- OLED display with scrolling playhead and live grid
- Audio output through LM386 amplifier and speaker

## Hardware in the video

- ESP32 DevKit V1
- SSD1306 128×64 OLED
- Tactile push buttons (8 step + 4 control)
- Potentiometer for BPM
- LM386 amp + salvaged desktop speaker
