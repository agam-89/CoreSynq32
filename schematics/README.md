# schematics

Circuit diagrams for CoreSynq32.

## Files

**`wokwi_diagram.png`** — Full system schematic covering the digital side: ESP32, SSD1306 OLED, 8 step buttons, 4 control buttons, and BPM potentiometer. Created in Wokwi.

**`386amp.kicad_sch`** — Analog audio output stage. ESP32 DAC (GPIO 25) → LM386 amplifier → 1000µF output cap → speaker. Designed in KiCad.
